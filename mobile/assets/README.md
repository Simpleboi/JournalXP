# Assets Directory

Place your app assets here:

## Required Files

- `icon.png` - App icon (1024x1024)
- `splash.png` - Splash screen (1284x2778 for iPhone 14 Pro Max)
- `adaptive-icon.png` - Android adaptive icon (1024x1024)
- `favicon.png` - Web favicon (48x48 or larger)

## Generating Assets

You can use Expo's asset generation:

```bash
npx expo-optimize
```

Or use online tools:
- [App Icon Generator](https://www.appicon.co/)
- [Adaptive Icon Generator](https://romannurik.github.io/AndroidAssetStudio/)
